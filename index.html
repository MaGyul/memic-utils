🛠️ 미밐 유저 스크립트 관리자

미밐의 불편함을 해결하는 통합 유저 스크립트

⚠️ 중요: 이 스크립트는 유저 스크립트 매니저가 필요합니다!
파일을 직접 실행하면 작동하지 않습니다.

📋 설치 방법

1단계: Tampermonkey 설치

브라우저에 Tampermonkey 확장 프로그램을 먼저 설치하세요:

🔧 Tampermonkey 다운로드

2단계: 스크립트 설치

Tampermonkey 설치 후 아래 버튼을 클릭하여 스크립트를 설치하세요:

📥 유저 스크립트 설치

3단계: 설치 확인

Tampermonkey 대시보드에서 "미밐 유저 스크립트 관리자"가 설치되었는지 확인하세요.

❓ 문제 해결

  • 스크립트가 작동하지 않는 경우: Tampermonkey가 올바르게 설치되었는지 확인
  • 설치 버튼이 작동하지 않는 경우: 브라우저에서 팝업을 허용했는지 확인
  • 기타 문제: 깃헙 이슈 페이지 방문
주의사항:
• 파일을 직접 더블클릭하여 실행하지 마세요
• 반드시 브라우저에서 설치 버튼을 통해 설치하세요
• Windows Script Host 오류가 발생하면 위 방법을 따라주세요
